# @muze-nl/metro-core

Small Metro core package. It contains the fetch-compatible client and the standard helper constructors:

- `client()` and `Client`
- `request()`
- `response()`
- `url()`
- `metroError()`

Higher-level API helpers, middleware, tracing, formdata helpers and the beginner-friendly global `metro` object live in `@muze-nl/metro` and related optional packages.
